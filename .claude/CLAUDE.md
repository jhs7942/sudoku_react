# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 언어 및 커뮤니케이션 규칙

### 기본 응답 언어
- **Claude와의 모든 소통**: 한국어
- **코드 주석**: 한국어로 작성
- **커밋 메시지**: 한국어로 작성
- **문서화 및 README**: 한국어로 작성

### 코드 표준
- **변수명/함수명**: 영어 (국제 개발 표준 준수)
- **클래스명/모듈명**: 영어
- **상수명**: 영어 (UPPER_CASE)

## 프로젝트 설정
- 계획은 항상 ./claude/plans/plan.md파일에 저장한다.
- 코드 구현 전 기능이 없는 프로토타입을 만들어서 디자인을 확인한다.
- 
- 코드 구현 후 오류는 없는지 테스트를 진행한다.

### 추가될 항목 (필요시)
- 빌드 및 실행 명령어
- 테스트 실행 방법
- 코드 아키텍처 및 구조
- 의존성 설정
- 개발 환경 설정
